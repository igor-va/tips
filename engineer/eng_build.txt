Files in the root folder and subfolder
	- develop.dwg  # Development file
	- manual.dwg  # Assembly instructions
	.pdf  # Various drawings for discussion
    - plan.odt  # The procedure for doing work
	- wiki.txt  # General technical information about the product or part being developed
	- tech.xlsx  # Specification, equipment, weight, length, strength
	- eng.docx  # Translations from Russian into English


Названия разрабатываемых деталей на сервере
	Detail Name v?  # Профиль Крышка v2
	Detail Name v? - Type  # Профиль Крышка v2 - Сечение
	Project Name - Detail Name v? - Type  # Профиль зажимной - Профиль Крышка v2 - Сечение


Названия разрабатываемых деталей во фьюжн
	Detail Name  # Крышка торцевая
	Detail Name Ver1  # Крышка торцевая Ver1
	Assembly Name  # Сборка
	Assembly Name Ver1  # Сборка Ver1
