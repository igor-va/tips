Files in the root folder and subfolder
	- develop.dwg  # Development file
	- manual.dwg  # Assembly instructions

	- eng.docx  # Translations from Russian into English


Названия разрабатываемых деталей на сервере

	Project Name - Detail Name v? - Type  # Профиль зажимной - Профиль Крышка v2 - Сечение
	Project Name - Date  # Поручень LED - 2021.05.05
	Project Name v? - Date  # Поручень LED v2 - 2021.05.05
	Project Name - Detail Name - Date  # Поручень LED - Крышка торцевая - 2021.05.05
	Project Name - Detail Name v? - Date  # Поручень LED - Крышка торцевая v2 - 2021.05.05
	Project Name v? - Detail Name - Date  # Поручень LED v9.2.2 - Крышка торцевая - 2021.05.05
