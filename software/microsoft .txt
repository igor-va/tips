Windows
	Добавить в path
		Алгоритм
			1. В строке "Поиск" выполните поиск: Система (Панель управления)
			2. Нажмите на ссылку Дополнительные параметры системы
			3. Нажмите Переменные среды
			4. В окне Изменение системной переменной (или Новая системная переменная) укажите значение переменной среды PATH		
		Ссылки
			https://nicothin.pro/page/windows-path
	Список исполняемых команд (не работает у меня)
		echo %PATH%
	Комманды
        Вернуть окно приложения вне экрана
            Нажмите Alt + Tab и клавишей Tab выберите нужное окно
            Затем нажмите Alt + Пробел + X, это развернёт окно на весь экран
            Затем окно можно переместить куда надо, отрегулировать размер или закрыть
        Копия активного окна
            Alt + PrintScreen
