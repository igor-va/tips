# Команды
CTRL+/  # Comment code
Double Shift  # Search everywhere for classes, files, tool windows, actions, and settings
Shift + F6  # Replacing the name in the entire project
Ctrl + F8  # Toggle the breakpoint
Ctrl + Space  # Code completion 
Ctrl + W  # Extend selection
Alt + LM  # Select a few mouse cursors
